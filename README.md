Power BI Sales Dashboard

Project Overview:
This project is an interactive sales dashboard built using Power BI.

Tools Used:
- Power BI
- Excel / CSV Dataset

Features:
- Total Sales, Profit, Quantity KPIs
- Sales by Category
- Top Products Analysis
- Region-wise filtering using slicers

Insights:
- Identified top-performing products
- Compared regional performance
- Built interactive dashboard
